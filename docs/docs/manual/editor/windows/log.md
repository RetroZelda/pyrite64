# Log

The **Log** window shows the editor and build output: status messages, warnings, and errors.
It is the first place to look when a build fails or something does not behave as expected. The
view auto-scrolls to the newest line as messages arrive.

## Toolbar

| Button | Description |
|--------|-------------|
| **Clear** | Empty the log. |
| **Copy** | Copy the entire log to the clipboard. |
| **Save** | Write the log to a timestamped `log_*.txt` file in the project folder. |
